Onion Search Directory

A list of Tor network search engines.

⚠️ Alert: Use the Tor Browser (preferably in a Tails or Whonix environment) to access these links. Never enter any personal details that could reveal your identity on these platforms.

DuckDuckGo (Onion)

The default search engine for the Tor browser. It provides tracking-free searches and displays dark web results alongside standard web results.
